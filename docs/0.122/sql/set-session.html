7.13. SET SESSION

Synopsis

SET SESSION name = expression
SET SESSION catalog.name = expression

Description

Set a session property value.

Examples

SET SESSION optimize_hash_generation = true;
SET SESSION hive.optimized_reader_enabled = true;
